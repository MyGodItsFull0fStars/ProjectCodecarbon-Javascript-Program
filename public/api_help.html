How do start?

  1. You have to create an organisation
  2. Then you have to create a team with the organisation id
  3. Next a project with the team id
  4. Then you have to create an experiment with the project id
  5. Next you write your experiment id in the file .codecarbon.config
  6. Now you have just to write the command codecarbon init and codecarbon monitor in the cmd
  7. You can see the data on the API Start Page
